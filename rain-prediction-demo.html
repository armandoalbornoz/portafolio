Project Overview

A production-ready ML pipeline that predicts rainfall using historical weather data from the Open-Meteo API. The system demonstrates modern MLOps practices with automated data versioning, experiment tracking, and model deployment.

Key Features

  • Modular, configuration-driven architecture for easy maintenance
  • ETL pipeline with PostgreSQL RDS for scalable data storage
  • Multiple model training with hyperparameter optimization
  • DVC for data and model versioning
  • MLflow for experiment tracking and model registry
  • Real-time predictions via Streamlit interface

Technical Highlights

  • Clean code architecture with separation of concerns
  • Automated data pipeline from API to database
  • Model comparison across multiple estimators
  • Cloud deployment on Streamlit Community Cloud
  • Version control for datasets and models
